## 초기 설정

초기 설정을 위해 터미널에 입력한 CLI 명령어는 다음과 같습니다.

1. react 설치

```
npm i create-react-app
```

2. react-router-dom 설치

```
npm i react-router-dom
```

3. styled-components, styled-reset 설치

```
npm i styled-components styled-reset
```

## 사용한 Node 모듈

- axios

```
npm i axios
```
